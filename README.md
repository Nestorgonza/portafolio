# Portafolio — Nestor Fabián González

Portafolio personal como desarrollador web. Sitio estático de una sola página, construido a mano con HTML, CSS y JavaScript — sin frameworks ni plantillas.

**🔗 Ver en línea:** [nestorgonza.github.io/portafolio](https://nestorgonza.github.io/portafolio/)

## Sobre mí

Desarrollador web con 6 años de experiencia en desarrollo y soporte técnico, especializado en administración, mantenimiento y actualización de sitios web. AWS Certified Cloud Practitioner. Integro herramientas de IA (Claude Code, ChatGPT, Gemini) en mi flujo de trabajo diario.

Actualmente desarrollo una aplicación de seguimiento ocular en Python para que personas con dificultades en la comunicación puedan controlar el computador con la mirada.

## Tecnologías del sitio

- **HTML5** semántico y accesible
- **CSS3** (variables, grid, flexbox, `prefers-reduced-motion`)
- **JavaScript** vanilla (sin dependencias)
- Tipografías de Google Fonts
- Desplegado con **GitHub Pages**

## Características

- 📱 Diseño responsive (móvil primero)
- ♿ Enfoque de accesibilidad: contraste adecuado, foco visible, respeto a la preferencia de movimiento reducido del sistema
- ⚡ Un solo archivo, carga rápida, sin build ni dependencias

## Ejecutar localmente

No requiere instalación. Clona el repositorio y abre `index.html` en el navegador:

```bash
git clone https://github.com/Nestorgonza/portafolio.git
cd portafolio
```
